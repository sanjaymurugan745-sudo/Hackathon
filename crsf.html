About CSRF

What is CSRF?

Cross-Site Request Forgery (CSRF) is an attack that forces authenticated users to submit unintended requests.

Why is this vulnerable?

This application doesn't use anti-CSRF tokens to verify that requests originate from legitimate forms.

Secure Solution

Implement CSRF tokens that change with each session:

// Server generates and validates unique token
app.use(csurf());

// Form includes the token
<input type="hidden" name="_csrf" value="<%= csrfToken %>">

// Server verifies token on submission
if (!validCSRFToken(req)) {
  return res.status(403).send('Invalid CSRF token');
}
